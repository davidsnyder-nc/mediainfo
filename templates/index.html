Cron Job Setup

To run this script daily via cron job, add the following line to your crontab:

0 6 * * * cd /path/to/your/project && python run_daily.py
This runs the sync every day at 6:00 AM. Adjust the time as needed. Use crontab -e to edit your cron jobs.
